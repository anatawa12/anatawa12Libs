anatawa12Libs
====

Overview

## Description

Anatawa12's Core Library for kotlin.

## Requirement

Kotlin 1.2.70
If you use maven or gradle, you don't have to care about kotlin.

## Install

### maven

add this to pom.xml

```xml
<dependencies>
    <dependency>
        <groupId>com.anatawa12</groupId>
        <artifactId>anatawa12CoreLibs</artifactId>
        <version>0.0.1</version>
    </dependency>
</dependencies>
```

### gradle

add this to build.gradle

```groovy
repositories {
    mavenCentral()
}

dependencies {
    compile "com.anatawa12:anatawa12CoreLibs:0.0.1"
}
```
